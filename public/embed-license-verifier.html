Embed the License Verifier Tool

Copy the code below to embed the Open Acre Contractor License Verifier on your website. The tool is fully responsive and works on all devices.

Basic Embed Code
<iframe src="https://openacre.co/tools/license-verifier/embed" width="100%" height="600" frameborder="0" allowtransparency="true" allow="payment" ></iframe>
Preview: Paste this code into your website's HTML editor or page builder. The tool will load and be fully functional immediately.
Customize the size:
width: Set to 100% for full width, or use a pixel value like 600px
height: Default is 600px. Adjust based on content length.
Example for Wordpress / Webflow / Squarespace

Most website builders support embedding iframes via "Custom Code" or "HTML Embed" blocks. Follow your platform's instructions for adding custom HTML, then paste the code above.

Note: Some page builders (like Webflow) require you to enable "allow" permissions for payment attribute. This is harmless and just gives the iframe permission to function properly.
Responsive Best Practices
Set width="100%" to make it fill its container.
For mobile, you may want to reduce height to 500px or 400px on screens <600px wide.
The tool automatically adjusts its layout for mobile — you just need to set the container size.
